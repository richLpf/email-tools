# MailTools

**MailTools** 是一个面向邮件营销与邮箱相关工作的**工具导航站**：集中展示验证、预热、外联、模板设计、送达检测等方向的常用工具，支持一键跳转官网，并可在站内提交需求或订阅更新。

---

## 网站能做什么

| 能力 | 说明 |
|------|------|
| **浏览与筛选** | 按分类（验证、预热、外联、设计与模板、送达）筛选，支持关键词搜索。 |
| **外链访问** | 每个工具提供官网链接，在新标签页打开（第三方服务与本站无隶属关系）。 |
| **详情抽屉** | 点击卡片查看：工具名称、作用说明、链接、分类、收费方式（免费 / 付费 / 部分免费）、简要使用方法。 |
| **中英文** | 导航栏可切换界面语言，便于中英文读者使用。 |

---

## 订阅与需求反馈

站点提供独立页面 **「需求与订阅」**（`request.html`），用于：

- **提交工具建议**：若列表中没有你需要的工具，可填写姓名、邮箱与描述，向我们反馈需求（当前为前端演示流程，实际对接后端或邮件服务需自行扩展）。
- **订阅更新**：切换到「订阅更新」后，可登记意向，以便在新工具收录或站点更新时收到通知（同样需要后续接入真实订阅渠道）。

使用前请按页面提示完成邮箱验证步骤（演示逻辑），提交按钮在验证通过后方可启用。

---

## 技术说明（简要）

- **纯前端静态站点**：工具数据与文案写在仓库内，无需数据库即可展示与构建。
- **构建栈**：Vite、Tailwind CSS、原生 JavaScript；包管理推荐使用 **pnpm**。
- **扩展维护**：新增工具、本地启动与构建命令见 **[GUIDE.md](./GUIDE.md)**。

---

## 快速开始

```bash
pnpm install
pnpm dev
```

浏览器访问开发服务器提示的地址（一般为 `http://localhost:5173`）。生产构建：

```bash
pnpm build
```

产物在 `dist/` 目录，可部署到任意静态托管环境。
